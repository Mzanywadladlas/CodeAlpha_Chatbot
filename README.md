README: Chatbot
Repository name: CodeAlpha_Chatbot
Title: CodeAlpha Basic Chatbot (Python)

Project Description:
This is a simple rule-based chatbot built in Python. The chatbot responds to basic user inputs using predefined responses.

Features:
1. Responds to greetings like hello and hi
2. Answers basic questions
3. Exit command support (bye)
4. Continuous conversation loop
5. Console-based chatbot
   
Technologies Used:
* Python
* Functions
* If-elif conditions
* Loops
* Input and output handling
  
How to Run the Project:
1. Open Command Prompt or Terminal
2. Navigate to the chatbot project folder
3. Run the command:
python chatbot.py

Example Commands:
** hello
** how are you
** bye

Concepts Demonstrated:
*** Conditional statements
*** Functions
*** Infinite loops
*** String comparison
*** User interaction
